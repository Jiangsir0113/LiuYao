# 六爻选股分析工具

基于六爻易学与技术分析相结合的 A 股辅助研究工具。通过时间起卦为每只股票生成独立卦象，结合 MA、MACD、RSI 等技术指标，输出综合结论供参考。

> **免责声明**：本工具仅供学习与研究用途，不构成任何投资建议。股市有风险，入市须谨慎。

---

## 功能特性

- **时间起卦**：以当前年月日时辰为基础，叠加股票代码数字之和，为每只股票生成独立的六十四卦卦象
- **六爻断卦**：自动推算六亲、六神、用神（妻财爻）旺衰，给出六爻层面的买卖结论
- **技术面分析**：获取近 60 日 K 线，计算 MA5/MA20、MACD、RSI，判断趋势方向
- **综合结论**：融合六爻结论与技术趋势，输出"强烈推荐 / 可以关注 / 谨慎观望 / 继续观望 / 暂不介入 / 不宜买入"六档评级
- **股票筛选**：按成交量、涨幅范围过滤全市场 A 股，快速锁定候选标的
- **三栏 GUI**：PyQt6 桌面应用，左侧筛选条件、中间候选列表、右侧详细分析，操作直观

## 快速开始

### 环境要求

- Python 3.10+
- Windows / macOS / Linux（网络需能访问新浪财经 API）

### 安装依赖

```bash
git clone https://github.com/Jiangsir0113/LiuYao.git
cd LiuYao
pip install -r requirements.txt
```

### 运行

```bash
python main.py
```

### 打包为 Windows 可执行文件

```bash
pyinstaller LiuYao.spec
```

打包完成后，可执行文件位于 `dist/LiuYao.exe`。

## 核心算法说明

### 时间起卦

以当前时间的年、月、日、时辰（十二时辰编号 1–12）为基础，叠加股票代码各位数字之和，确保同一时刻不同股票起出不同的卦。

```
上卦索引 = (年 + 月 + 日 + 代码数字和) mod 8
下卦索引 = (年 + 月 + 日 + 时辰 + 代码数字和) mod 8
动爻位置 = (年 + 月 + 日 + 时辰 + 代码数字和) mod 6
```

最终通过六十四卦映射表查得卦名（如"火天大有"）。

### 六爻断卦

1. **六亲**：根据日干五行与各爻所属宫卦五行的生克关系，推算兄弟、子孙、妻财、官鬼、父母
2. **六神**：从初爻起，依次配以青龙、朱雀、勾陈、腾蛇、白虎、玄武
3. **用神**：股票问题以妻财爻为用神
4. **旺衰**：日干生用神则旺，日干克用神则衰；动爻为用神时额外加权
5. **结论**：用神旺且动爻有利 → 宜买；用神衰且动爻不利 → 不宜；其余 → 观望

### 技术面分析

- **数据来源**：新浪财经 API（`akshare.stock_zh_a_daily`），复权方式为前复权
- **指标计算**：纯 pandas/numpy 实现，不依赖 pandas-ta
  - MA5 / MA20：简单移动平均
  - MACD：EMA(12) − EMA(26)，信号线 EMA(9)
  - RSI：14 日相对强弱指数
- **趋势判断**：MA5 > MA20 且 MACD > 0 且 RSI 适中 → 上涨；反之 → 下跌；其余 → 横盘

### 综合结论映射

| 六爻结论 | 技术趋势 | 综合结论 |
|---------|---------|---------|
| 宜买    | 上涨    | 强烈推荐 |
| 宜买    | 横盘    | 可以关注 |
| 宜买    | 下跌    | 谨慎观望 |
| 观望    | 上涨    | 谨慎观望 |
| 观望    | 横盘/下跌| 继续观望 |
| 不宜    | 上涨    | 暂不介入 |
| 不宜    | 横盘/下跌| 不宜买入 |
